BabyPhone

Zwei Smartphones. Eine direkte WebRTC-Verbindung. Auf dem BabyPhone wählst du, ob Audio oder Video gesendet wird. Das ParentPhone hört oder sieht nur den Stream vom Baby.

1. Rolle wählen

Jedes Smartphone startet mit einer klaren Rolle.

2. Phones matchen

Nutzt auf beiden Phones denselben Code. Du kannst ihn hier erzeugen oder per Link weitergeben.

Erst Rolle wählen, dann auf beiden Phones denselben kurzen Code eingeben und auf beiden „Verbinden“ tippen.

3. Was soll das Baby senden?

Diese Auswahl gilt nur für das BabyPhone. Das ParentPhone empfängt automatisch den gewählten Stream.

Bereit zum Starten

Die Phones haben sich gefunden, aber WebRTC konnte keinen direkten Pfad aufbauen. In diesem Netz ist wahrscheinlich ein TURN-Server nötig.
